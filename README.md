# twitter-clone
Twitter clone utilizing the following stack:
- Typescript
- Node.js
- PostgreSQL
- Redis
- Object storage for media (specifics undecided)
- Express.js
- React.js
---
### Features
1. Create tweets
3. Follow others
4. View feed
These will be tackled sequentially and built to allow seamless additions in the future.
